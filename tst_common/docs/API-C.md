# 断言
## assert_true
```C
// 原型：assert_true(expr)
// 功能：断言表达式返回真
// 参数：
//   expr -- 表达式
// 返回值：无
// 示例：
assert_true(1 == 2);
assert_true(a >= b);
```

## assert_false
```C
// 原型：assert_false(expr)
// 功能：断言表达式返回假
// 参数：
//   expr -- 表达式
// 返回值：无
// 示例：
assert_false(1 == 2);
assert_false(a >= b);
```
